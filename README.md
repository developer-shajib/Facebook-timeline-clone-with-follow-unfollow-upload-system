## Run Locally

Clone the project or download it

```bash
  git clone https://github.com/developer-shajib/Todo-with-Redux
```

### Go to the project directory:

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```

## Tech Stack

**Client:** Ejs Template Engine

**Server:** Node, Express, MongoDb
